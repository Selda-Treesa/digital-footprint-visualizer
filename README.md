# 🔍 Digital Footprint Visualizer

A web app that shows how traceable a username is across multiple platforms using public data.

## 🚀 Features

* Checks 10 platforms (GitHub, Reddit, npm, etc.)
* Calculates a traceability score
* Shows reasons behind the score
* Extracts interests from public bios
* Visual charts (radar + breakdown)

## ⚠️ Disclaimer

This tool only uses publicly available data. Do not misuse it.

## ▶️ Run locally

pip install flask flask-cors requests
python app.py

Then open: http://127.0.0.1:5000/
